🚀 核心能力详解

1. 数据产品目录

精选推荐、Markdown 详情、权限申请/审批、资产全景（修复调用量统计）、批量发布与冗余清理。注意：审批菜单不再默认全员可见。

2. 数据源 AI 摸排

串行任务、可中断、全量重跑；表/字段 AI 画像（术语、标签、置信度）；「查看画像」弹层浏览；画像直导元数据导入。

3. SQL Lab 全栈

• 结果：表格/图表/透视/统计/行转列、分页、EXPLAIN、同步+异步导出

• 探索：表探索器（标签/收藏/已摸排）、侧栏表详情默认数据预览

• AI：流式分析、中断/复制/重试、会话存档、生成反馈赞踩

4. SQL Server + 资源治理

SQL Server ODBC 适配（Docker 镜像内置驱动）；资源配置版本历史与回滚；站内/钉钉/企微变更通知；MCP Server 开关。

5. 体验与其他

管理后台移动端适配；品牌/版权个性化；元数据向量一键同步；滚动条抖动等体验修复。
