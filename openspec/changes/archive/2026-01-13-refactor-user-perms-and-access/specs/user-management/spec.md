# 用户管理规格变更 (Spec Delta)

## MODIFIED Requirements

### Requirement: 用户创建与编辑 UI
系统 **必须 (MUST)** 提供一个用于创建和编辑用户的弹窗。
-   **已修改**：弹窗 **必须 (MUST)** 组织为标签页：“基础信息”和“权限分配”。
-   **已修改**：“权限分配”标签页 **必须 (MUST)** 仅在选择的角色为“普通用户” (user) 时可见或激活。
-   **未变更**：管理员用户通常拥有全量访问权限，因此不适用具体的资源权限分配或对其隐藏。

#### Scenario: 管理员编辑普通用户
-   **假设** 我是用户管理页面上的管理员
-   **当** 我点击一个普通用户的“编辑”按钮
-   **那么** 我会看到一个包含两个标签页的弹窗：“基础信息”和“权限分配”
-   **并且** 我可以在“基础信息”中修改用户名和备注
-   **并且** 我可以在“权限分配”中修改资源授权

#### Scenario: 管理员创建管理员用户
-   **假设** 我是以管理员身份创建一个新用户
-   **当** 我选择“管理员”作为角色
-   **那么** “权限分配”标签页将被隐藏或禁用
